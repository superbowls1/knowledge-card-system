# Knowledge Card System

## Overview
This project is a cloud-based web app that lets users create and search knowledge cards.

Each card has:
- Title
- Description
- Image (stored in AWS S3)

---

## Features
- Create cards with text and image
- Store data in database
- Store images in AWS S3
- View all cards
- Search cards by keyword
- Delete cards (removes from S3 and database)
- Deployed on AWS EC2

---

## Tech Used
- Python
- Django
- AWS EC2
- AWS S3
- SQLite
